Кнопки быстрого вызова

Рис. 1 Кнопки быстрого вызова на графике

В терминале Trade# ряд действий можно выполнить, не переходя по пунктам меню и не пользуясь "горячими" сочетаниями клавиш. Эти действия вынесены на графике в виде пиктограмм. Набор пиктограмм настраивается в окне Параметров (меню Настройки | Параметры), вкладка "Быстрый вызов":

Рис. 2 Настройка пиктограмм быстрого вызова

Для того, чтобы кнопка (пиктограмма) отображалась на всех графиках, в соответствующей строке таблице должна быть выставлена "галочка".

Автопрокрутка граф.

Включить / отключить автоматический скроллинг графика.

Архив котировок

Открыть окно архива котировок - окно показывает содержимое локального "кэша" котировок, управляет актуализацией ("подкачкой" котировок с сервера) и формированием списка "избранных" инструментов.

Быстро купить / быстро продать

По нажатию на кнопку появляется список выбора объема сделки. Если в списке отсутствует нужный объем, список можно редактировать, выбрав вариант - Другое значение. После выбора объема будет открыто окно подтверждения совершения торговой операции. Этот шаг можно пропустить - в этом случае, размещение нового рыночного ордера будет произведено всего двумя кликами мыши! Соответствующая опция доступна в меню Настройки | Параметры, вкладка Торговля (Быстрый вызов - подтверждение ордера).

Избранные индикаторы

Добавление технического индикатора на графике также можно произвести парой кликов мыши. В списке показываются "избранные" индикаторы - те, что отмечены "звездочкой" в окне выбора индикатора (см ниже).

Индикаторы

Открывает окно добавления технического индикатора на график. В окне можно настроить список избранных индикаторов:

Рис. 3 Окно добавления индикатора - настройка "избранного" списка

Новый ордер

Открывает стандартный диалог размещения нового ордера (рыночного или отложенного).

Сделки по инструменту

Открывает список открытых позиций (по торгуемому инструменту выбранного графика). Вверху списка показана суммарная позиция по инструменту. Кликом левой кнопки мыши открывается окно редактирования / закрытия позиции.

Рис. 4 Список открытых позиций

Синхронизировать историю

Открывает окно синхронизации (с сервером) локального кэша котировок по выбранному инструменту - инструменту графика. Можно загрузить с сервера больший интервал истории котировок, чем доступен на текущем графике.

Статистика по инструменту

Открывает таблицу сделок по текущему торгуемому инструменту, во многом аналогичную списку "сделок по инструменту" (выше). В дополнение присутствуют:

  • Закрытие всех ордеров по инструменту одной кнопкой "Закрыть все".
  • "Связывание" ордеров кнопкой "Связать". Подразумевается установка общего SL / TP ордера для выбранных сделок.
  • Возможность "закрепить" окно ордеров на графике кнопкой в верхнем правом углу окна.

Рис. 5 Таблица открытых позиций

Для "связывания" ордера должны быть предварительно отмечены красной "звездочкой" в левом столбце таблицы. Уже "свзанные" ордера разбиты на группы, где, для всех ордеров каждой группы, установлен общий SL / TP ордер. Ордер каждой группы отмечен "звездочкой" с одним номером.
