Окно выбора индикаторов

Вызывается при добавлении / редактировании технических индикаторов в окне графика.

Рис. 1 Окно индикаторов графика

Индикаторы, уже добавленные на график, перечислены в таблице. Добавить индикатор можно кнопкой + в левом верхнем углу окна. Убрать индикатор с графика можно кнопкой x вверху окна либо в крайнем правом столбце таблицы. Редактируется индикатор двойным щелчком левой кнопкой мыши в таблице либо нажатием кнопки с изображением карандаша, вверху окна.

Избранные индикаторы

Во втором столбце таблице некоторые индикаторы могут быть отмечены звездочкой (*). Эти индикаторы - "избранный" список. Индикаторы из такого списка могут быть быстро добавлены на график. Для того, чтобы добавить или исключить индикатор из списка "избранных", достаточно нажать левой кнопкой мыши во втором столбце таблицы.

Графа "Ссылки"

Некоторые индикаторы могут строиться от данных других индикаторов. Например, индикатор "дивергенции". Для таких индикаторов в поле "Ссылки" перечислены индикаторы, на данные которых они ссылаются, по номерам (первая графа таблицы).

Добавление индикатора

Нажатием на кнопку + открывается окно выбора индикатора:

Рис. 2 Окно добавления индикатора

Индикаторы упорядочены в виде раскрывающегося по категории списка либо таблицы. Таблицу можно упорядочить по любому столбцу / по нескольким столбцам. Двойным щелчком по строке списка / таблицы, либо выбором левой кнопкой мыши и нажатием кнопки "Выбрать" индикатор добавляется на график.
В левом столбце таблицы, равно как и в таблице индикаторов (выше), можно добавить / исключить индикатор из "избранного" списка.

Настройка индикатора

Двойным щелчком либо кнопкой с изображением карандаша открывается окно настроек индикатора:

Рис. 3 Настройки индикатора

Настройки каждого индикатора разбиты по группам. Для того, чтобы изменить значение свойства, в соответствующее поле вписывается текст (число), значение выбирается из списка либо устанавливается в отдельном окне. Некоторые свойства стоит рассмотреть отдельно.

Создавать панель отрисовки

Вместе со свойствами Отображать панель отрисовки, Отображать на графике определяет, будет ли индикатор рисоваться:

  1. В окне основного графика (графика котировки): "Отображать на графике" - true, "Панель отрисовки" - Курс
  2. В собственном окне: в поле "Панель отрисовки" выбрать название настраиваемого индикатора. В полях Создавать панель отрисовки, Отображать панель отрисовки автоматически установится значение - true
  3. В отдельном окне другого индикатора: аналогично п. 2, но в поле "Панель отрисовки" выбрать название окна другого индикатора

Серии источников

Индикатор может быть построен не только от значений котировки рыночного курса, но и от значений других индикаторов. В приведенном примере индикатор "Дивергенции" строится от двух источников - двух серий данных. Индикатор указывает срелкой два последовательных экстремума одного знака, направление "роста" или "снижения" которых не "подтверждается" трендом второй серии.

Оба этих источника (серии) данных должны быть указаны в поле Серии источников - отмечены в выпадающем списке.

Рис. 4 Серии - источники данных

В приведенном на рисунке примере индикатор должен искать дивергенции между значениями индикатора "Скользящая средняя" и средней линией (BMA, Bollinger Moving Average) индикатора "Ленты Боллинджера" (пример, разумеется, не имеет практической ценности). Скользящая средняя формирует максимумы / минимумы, средняя линия индикатора "Ленты Боллинджера" (скользящая средняя с другими параметрами) должна "подтверждать" или "опровергать" эти экстремумы. Первой должна быть указана "Скользящая Средняя", однако в списке она идет после "Лент Боллинджера".
Для задания правильного порядка серий-источников данных нужно выбрать "Скользящая Следняя/-/СС" и клавишей курсора "Вверх" переместить "Скользящую Среднюю" выше "Лент Боллинджера".
