Урок 2

Online-чат

Пользователи торгового терминала TradeSharp могут обсудить технические вопросы, связанные с терминалом, в online-чате.

Чат в Trade# - средство мгновенного обмена сообщениями. Чат имеет централизованную архитектуру: всех чат-клиентов обслуживает единственный чат-сервер. Компания TradeSharp поддерживает чат-сервер, реализующий собственный протокол обмена сообщениями.

Чат организует обмен сообщениями для поддержания коллективного общения. Все участники чата вовлечены в единый диалог: сообщения одного рассылаются всем остальным. Для организации нескольких независимых диалогов используется деление на комнаты чата. Комната – способ группирования участников чата. Участники комнаты – это некоторое подмножество множества всех участников чата. Подмножества участников комнаты в общем случае пересекаются. Существуют комнаты без участников. Сообщения, адресованные в комнаты, рассылаются только участникам этой комнаты.

Пользовательский интерфейс чата

Чат-клиент встроен в терминал. Чтобы открыть окно чата нужно нажать на кнопку «Окно чата» в правом нижнем углу терминала.

Рис. 1 Кнопка запуска чат-клиента

Чат использует идентификацию терминала: логин, используемый для входа в терминал, является также логином для входа в чат.

Рис. 2 Главное окно чата

Окно разделено на 2 части: левая часть отведена под поля для приема и отправки сообщений, правая часть – для отображения списка участников чата. В нижней части окна находится строка состояния. В ней отображается ход выполнения запроса к серверу.

Чтобы стать участником комнаты нужно выполнить вход в нее. Для этого нужно нажать на кнопку «Комнаты…» или выбрать ссылку «Комнаты…».

Ссылка «Комнаты…» в главном окне чата будет доступна, если ни в одну из комнат вход не осуществлен. Если вход в чат не выполнен (если отключен автоматический вход – см. раздел «Настройки»), то вместо ссылки «Комнаты…» будет доступна ссылка «Вход».

Рис. 3 Главное окно чата при невыполненном входе

Рис. 4 Окно со списком комнат

При выборе какой-либо комнаты в поле описание будет выведено описание данной комнаты, заполненное ее владельцем. Для создания новой комнаты необходимо нажать кнопку «Создать…» и заполнить поля формы.

Рис. 5 Окно создания комнаты

Владелец комнаты – участник чата, наделенный правами на изменение описания, приветствия и пароля для входа в комнату. Владельца может менять только участником, наделенным правом администратора. В противном случае владельцем назначается тот, кто создает комнату.

Если комнату не будет ни одного пользователя в течение 15 минут, она будет автоматически удалена сервером по причине неактивности. Автоматическое удаление можно отключить, выставив флаг «не удалять из-за неактивности». Время, когда будет удалена комната можно увидеть, нажав на кнопку «Изменить…» в окне «Комнаты».

Рис. 6 Окно редактирования комнаты

Если комната создается не администратором, то поле «Владелец» и флаг «не удалять из-за неактивности» недоступны.

После нажатия на кнопку «Войти» окно «Комнаты» закроется, а в левой части главного окна появится соответствующая вкладка с сообщениями.

Рис. 7 Вкладка с сообщениями комнаты

Область (1) отображает сообщения, отправленные участниками этой комнаты. Область (2) предназначена для ввода собственного сообщения для последующей отправки по кнопке «». Кнопка «» используется для выхода из комнаты.

Рис. 7 Внешний вид сообщений

Сообщения чата можно разделить на 5 типов: собственные сообщения (1), сообщения владельца (2), прочие сообщения (3), уведомления (4) и сообщения об ошибках.

Первые 3 типа сообщений диалоговые, они вводятся участниками чата. Уведомления – недиалоговые сообщения, исходящие от сервера; различают 2 их вида: о входе и выходе из комнаты.

Настройка чата

Внешний вид сообщений чата настраивается из окна «Настройка чата», которое можно открыть, нажав на кнопку «Настройки…».

Рис. 8 Окно настройки чата

Помимо шрифта и цвета текста этим сообщений можно настроить также формат представления даты и времени, видимость вкладки лога чата и признак автоматического входа в чат.

Лог чата используется для просмотра запросов к серверу и ответов на них. Лог открывается в отдельной вкладке.

Автоматический вход в чат выполняется при авторизации пользователя в терминале без участия пользователя. При отключенном автоматическом входе вход и выход из чата осуществляется вручную по кнопке «Вход»/«Выход».
