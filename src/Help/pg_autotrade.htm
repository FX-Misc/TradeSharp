Настройки автоматической торговли

Терминал TradeSharp предлагает опцию автоматической торговли по сигналам, включающую гибкие настройки риск-менеджмента. Для включения / отключения автоматической торговли кликните по соответствующей колонке во вкладке подписки на торговые сигналы (меню Торговля > Сигналы, вкладка Подписка):

Рис. 1 Настройки получения торгового сигнала

Откроется окно управления автоматической торговлей по сигналу. NB: фактически, пользователь подписывается либо отключает опцию автоматической торговли на сервере торговой платформы. Если же пользователь настроил автоматическую торговлю, но затем осуществил вход с другого торгового терминала (на другом ПК либо после новой инсталляции терминала TradeSharp без сохранения старых настроек), в таблице подписки на торговые сигналы появятся предупреждающие отметки. По нажатию на отметке либо в столбце "Торговать" будет открыто окно настройки автоматической торговли.

Рис. 2 Окно настройки автоматической торговли

Торговать автоматически

Включить - отключить автоматическую торговлю.

Макс. плечо

Максимальное суммарное торговое плечо, допустимое для данного торгового сигнала.
Пример: текущий баланс торгового счета получателя сигнала равен 16 510 USD. Максимально разрешенное значение торгового плеча по сигналу задано равным 3.5.
По данной категории торгового сигнала открыты позиции:

BUY10 000GBPUSD
BUY10 000GBPUSD
SELL20 000USDCHF

Планируется торговая операция - SELL 60 000 GBPUSD. В случае совершения сделки экспозиция по валютной паре GBPUSD составит
|10 000 + 10 000 - 60 000| * 1.6105 / текущий курс USDCHF / = 64 420 USD.
Суммарная экспозиция в этом случае будет равна (64 420 /GBPUSD/ + 20 000 /USDCHF/) = 84 420 USD.
Новое торговое плечо составит 64 420 / 16 510 = 3.9, что превышает предельно допустимое значение (3.5).
Вход в рынок не будет произведен.

Процент плеча

Множитель (%) для торгового плеча по сделке. Пример:
Значение параметра = 150 (%).
Трейдер-автор сигнала открывает позицию BUY 50 000 EURUSD, текущий баланс его торгового счета 81 000 USD.
Плечо для позиции равно (50 000 * 1.3700 /курс EURUSD/) / 81 000 = 0.85.
Плечо для сделки получателя сигналов равно (0.85 * 150%) / 100% = 1.27.
Текущий баланс счета получателя торговых сигналов равен 11 000 USD.
Для выхода в рынок будет выделено 11 000 * 1.27 = 14 000 USD, объем входа будет расчитан как 14 000 / 1.3700 = 10 185 EUR и округлен до 10 000 EUR.

Хедж. ордера

Если флаг (встречные ордера) не выставлен, открытие позиции, противоположной по знаку уже открытым по данному торговому сигналу и данному активу позициям запрещено (запрещается иметь одновременно покупки и продажи по одному торговому сигналу по одному активу).

Magic

Идентификационный номер, присваиваемый торговым ордерам по данному торговому сигналу. Может использоваться для автоматизации торговли (скрипты, торговые роботы).

Фиксированный объем

Если флаг "фикс. объем" включен объем входа в рынок независим от расчетного плеча и равен сумме, заданной в поле справа.

Округление объема

Рассчитанное значение объема входа округляется по правилу, выбранному в поле (справа).
Пример: расчетный объем входа равен 17 800 ед. (например, USDCHF).
Минимальное значение объема входа: 10 000 ед. Шаг объема: 5 000 ед.

  • Тип округления - ближайшее. Результирующий объем: 20 000 ед.
  • Тип округления - вниз. Результат: 15 000 ед.
  • Тип округления - вверх. Результат: 20 000 ед.

Минимальный объем, шаг объема

Значения используются для округления объема входа, с тем, чтобы он стал кратным величине шаг объема, но не менее значения минимальный объем (см выше).
