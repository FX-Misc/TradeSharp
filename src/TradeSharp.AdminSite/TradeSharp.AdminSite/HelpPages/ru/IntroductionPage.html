Введение

Trade# server administration представляет собой многофункциональное клиентское приложение с графическим web интерфейсом, которое предназначено для администрирования торгового терминала Trade# и сочетает в себе широкий спектр функциональных возможностей необходимых администратору.

Trade# server administration разделён на два модуля: «Администрирование» и «Менеджмент».

Модуль «Администрирование» предоставляет инструменты для управления службами терминала (например, сервис котировок, служба торговых сигналов, чат - сервер и др.), обзор количества активных сеансов, зарегистрированных пользователей и др.

Функции модуля «Менеджмент»:

  • настройки торгуемых инструментов;
  • управление счетами, настройка их параметров;
  • управление группами счетов;
  • настройка и формирование отчётов;
  • другие дополнительные функции.
