Настройки

На странице настроек администратору предоставляется функционал для добавления, редактирования и настройки различных торговых инструментов, предоставляемых пользователям терминала.

Базовые активы

На странице «Базовые активы» предоставлен функционал для обзора, добавления и редактирования активов (например, валют), которые будут использоваться в терминале для торговли. Все активы должны иметь уникальное название.

Инструменты.

На странице инструменты («Таблица SPOT») администратор может добавлять и редактировать торговые инструменты (валютные пары и т.п.), которые будут доступны пользователям терминала.

Для редактирования торгового инструмента необходимо кликнуть соответствующей по ссылке в столбце «Название».

Метаданные.

На странице «Метаданные» администратору предоставляется функционал для тонкой настройки терминала Trade#. В качестве примера, на следующем рисунке приведены настройки выходных и праздничных дней («нерабочие дни», когда торговля не осуществляется). Среди них Новый год и новогодние праздники, рождество и др.

Все метаданные разбиты по категориям. Кнопка «Добавить», расположенная над всеми таблицами, добавляет новую категорию.

После добавления новой категории, на странице появляется соответствующая таблица, в которую уже можно добавлять отдельные записи (параметры). Для этого нужно кликнуть кнопку, находящуюся в «подвале» соответствующей таблицы.

Торговые сигналы.

Функционал торговых сигналов предоставляет клиентский терминал Trade#. Некоторые действия доступны только администратору (менеджеру) системы на странице управления сигналами.

Как это делается, подробно описывается на этой странице.

Администратор может производить следующие действия:

  • Добавить новый торговый сигнал. Это делается при помощи кнопки над таблицей.
  • Редактировать торговый сигнал. Для этого нужно кликнуть соответствующую ссылку в столбце «Название».
  • Удалить торговый сигнал. Для этого нужно кликнуть соответствующую ссылку в столбце «Удаление».
