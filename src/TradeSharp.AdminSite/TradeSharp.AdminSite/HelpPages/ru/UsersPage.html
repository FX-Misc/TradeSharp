Пользователи

На странице «Администрирование → Пользователи» может просмотреть справочную информацию о пользователях терминала, зарегистрированных на текущий момент и количество активных на текущий момент сеансов.

Географическое положение пользователей вычисляется по их IP адресу. Вследствие этого, сведения о географическом положении являются приблизительными.
